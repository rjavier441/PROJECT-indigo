### How do I get set up? ###
* Setup
   * Currently only tested/played on Linux
   * Install SFML (I'm using v2.3.2 in Ubuntu Linux). For help, see http://www.sfml-dev.org/tutorials/2.3/
   * Install a C++ compiler (in Linux, I use g++ v4.8.4)
